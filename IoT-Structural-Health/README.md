# 📡 IoT-Structural-Health

Real-time monitoring of structure integrity using distributed sensor networks and edge computing.

## 🚀 Getting Started
- `sensor_handler.py`: Simulates IoT data ingestion from accelerometers and detects damage events using vibration analysis (RMS).

## Usage
```bash
python sensor_handler.py
```

## Overview
Real-time monitoring system using **Sensor-Fusion** and **Early Warning Systems**.

### Key Features
- **Data Fusion:** Processing data from accelerometers and strain gauges.
- **Alerts:** Threshold analysis for structural fatigue and damage detection.
